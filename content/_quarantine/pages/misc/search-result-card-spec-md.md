\# Search Result Card Spec

Each result appears as a compact card.

\## Required elements

\- Title (clickable)

\- Type label (Constitution / Era / Event / Framework / Community / Explainer / Source)

\- Era badge (if applicable)

\- Date / date range (if applicable)

\- 1â€“2 line excerpt

\- Tags row (clickable)

\- Status + last updated (small, subdued)

\## Ordering

\- Exact title matches first

\- Then tag matches

\- Then text matches

Within each group:

\- Constitutions pinned above other types if explicitly searched

\- Events sorted by date

\- Other pages by relevance

\## Interaction

\- Click title: open page

\- Click tag: open tag index for that tag

\- Click era: open era page
